# Trackyo — AI Expense Tracker

> **“Track Smart. Spend Wise. That’s Trackyo.”**

Trackyo is a production-ready, full-stack personal finance and automated expense tracking platform designed for Indian users. The system automates personal bookkeeping by scanning receipts with client-side OCR, simulated bank transaction SMS reading, and auto-categorization using Google's **Gemini AI API**, all wrapped inside a sleek, premium fintech dashboard with custom visual themes.

---

## 🚀 Core Features

1. **User Authentication**: Secure JWT-based registration and login, persistent sessions, forgotten password simulations, and custom visual preferences (Currency & Theme).
2. **Home Dashboard**: Total balance summary, monthly spent limits, weekly expenditures, interactive area/pie charts, and personalized financial tips generated by **Gemini AI**.
3. **SMS Parser Simulator**: Automatically extracts amounts, merchant names, categories (e.g. Swiggy → Food, Uber → Travel), and bank details from Indian transactional alerts (SBI/HDFC/etc.) to automatically create logs.
4. **Receipt Scanner (OCR)**: Integrates client-side **Tesseract.js** to recognize receipts text and pre-fill expense forms instantly using AI.
5. **Budgets & Limit Alerts**: Track category limits with color-coded gauges (Green < 85%, Amber >= 85%, Red > 100% Blown budget warnings) and log custom in-app push alert banners when thresholds are breached.
6. **Wishlist & Milestones**: Setup specific targets (e.g. MacBook Pro, Goa Holiday), make custom piggy bank deposits, and trigger completion congratulations.
7. **Themes Customization**: Four premium custom visual styles:
   - **Dark Slate (Default)**: Modern slate obsidian with glowing teal/indigo indicators.
   - **Light Mode**: Ultra-clean cool gray glass borders.
   - **Cyber Neon**: Cyberpunk theme with electric pink and green glowing boundaries.
   - **Terminal Mono**: Minimalist, retro high-contrast monochrome terminal look.
8. **Auditing Admin Panel**: Dedicated metrics dashboard details system users registered, global transaction volumes, average expenditure costs, and live transaction grids.

## 🛠️ Installation & Setup

### Prerequisites
- Node.js (v18+)
- MongoDB (Running locally or a MongoDB Atlas URI link)

### 1. Database and Environment variables configuration
Navigate to `backend/.env` and ensure the parameters are configured:
```env
PORT=5000
MONGODB_URI=mongodb://127.0.0.1:27017/trackyo   # Fallback local connection
JWT_SECRET=trackyosecretjwt12345!@#
GEMINI_API_KEY=                                 # Paste your Google Gemini Key here
```
> [!NOTE]
> *If `GEMINI_API_KEY` is left blank, Trackyo automatically switches to a high-fidelity local keyword and rules-based heuristics categorization engine that mirrors the AI behavior perfectly for academic demonstrations!*

### 2. Seed Mock Database Logs
Populate the database with rich transactional history, active budgets, wishlist targets, and notifications:
```bash
cd backend
npm run seed
```
**Default seeded accounts details for testing:**
- **Rahul Sharma** (Normal User - extensive history): `rahul@trackyo.in` / `password123`
- **Priya Patel** (Normal User - blown budget alert): `priya@trackyo.in` / `password123`
- **Admin Master** (Super Admin account): `admin@trackyo.in` / `adminpassword`

### 3. Running Backend Server
Start the development server using nodemon:
```bash
cd backend
npm run dev
```

### 4. Running Frontend React Client
```bash
cd frontend
npm run dev
```
Open [http://localhost:5173](http://localhost:5173) in your browser to experience Trackyo!

## 🖥️ Verification Walkthrough

1. **Multi-Theme Swap**: Log in and toggle the top navbar dropdown selector between *Dark*, *Light*, *Neon*, and *Minimal* styles to verify custom colors apply smoothly.
2. **AI SMS Reader**: Click **Simulate Bank SMS** on the dashboard, load a preset template (e.g. *Swiggy Chicken Biryani Rs 450 HDFC*), click **AI Parse SMS** to extract merchant/amount details, and hit **Confirm** to record.
3. **Receipt Scanner**: Click **Scan Receipt with OCR**, select a mockup receipt template (like Zomato or Zara), and hit **Run AI OCR Scans** to see Tesseract extract details and the backend fill in values.
4. **Budget limit warnings**: Navigate to **Budgets Limit**, set your monthly spent cap to Rs. 2000, and add an expense exceeding it to trigger an alert.
5. **CSV Ledger Export**: Go to **Transactions**, click **Export CSV** to instantly download your transaction history as a fully structured spreadsheet.
